Universal Translator

MakeOHI/O 2025

Myself and a team of three others built the Universal Translator at MakeOHI/O 2025, a 24 hour makeathon at The Ohio State University. It is a real time sign language interpretation tool that uses Python and OpenCV to recognize ASL gestures and translate them into any selected language. We routed the translated text to an Arduino powered LCD, creating an easy communication bridge between ASL users and non ASL users.
